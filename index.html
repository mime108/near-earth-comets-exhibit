Star Field Map of Comets

This visualization presents near-Earth comets as glowing orbs in a cosmos. The orbs are scattered like stars in the night sky, creating a magical celestial scene.

Design Elements:

  • Marks: Circles (orbs) with radiating starburst lines
  • Channels:
    • Position (X): Perihelion distance (q) - how close the comet gets to the Sun
    • Position (Y): Eccentricity (e) - how elliptical the comet’s orbit is
    • Size: Minimum Orbit Intersection Distance (MOID) - smaller MOID (closer to Earth) creates larger orbs
    • Opacity: MOID - closer comets appear brighter
    • Color Gradient: Based on MOID values, creating visual separation between different comets
  • Interactions:
    • Hover: Reveals comet name via tooltip
    • Click: Highlights the selected comet and connects it to comets with similar perihelion years (within 5 years), displaying a detailed information card

Custom Visualization Category: Modded
This visualization modifies a standard scatter plot by adding glowing orbs, radiating starburst lines, and interactive connections. It retains the core structure of a scatter plot, enhancing it with visual effects and interactions to highlight cometary relationships.

The Spiral of Comets

This visualization renders near-Earth comets as a swirling spiral galaxy, with each comet as a curved arc orbiting the Sun, revealing orbital patterns and relationships.

Design Elements:

  • Marks: Curved arcs following logarithmic spiral paths
  • Channels:
    • Radius: Perihelion distance (q) - distance from the center represents proximity to the Sun
    • Arc Length: Orbital period (P) - longer arcs for comets with longer orbital periods
    • Thickness: Minimum Orbit Intersection Distance (MOID) - thicker lines for comets that pass closer to Earth’s orbit
    • Color: Eccentricity (e) - color groups comets by how elliptical their orbits are (blue for low, gold for medium, orange for high)
  • Interactions:
    • Hover: Highlights the arc and displays comet information
    • Click: Transitions to an orbital animation showing the comet’s path around the Sun, with an information dashboard displaying key orbital parameters

Custom Visualization Category: Novel
Rather than using standard circular orbits or scatter plots, it creates a unified, galaxy-like spiral that transforms temporal and spatial data into a cohesive visual structure. The design is inspired by artistic data visualizations like those created by Giorgia Lupi from this article https://giorgialupi.com/bulletin-of-the-atomic-scientists
